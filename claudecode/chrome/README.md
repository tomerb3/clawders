    Here's everything you can do with the kimi-webbridge skill controlling your real Chrome:

    Navigation
    - Open any URL
    - Open in a new tab
    - Find and switch to existing tabs
    - Close tabs or entire session

    Interaction
    - Click any element (buttons, links, anything)
    - Type/fill text into input fields, textareas, and rich-text editors (ProseMirror, Lexical, Slate)
    - Press keys (Enter, Tab, Escape, Arrow keys, etc.)
    - Upload files to <input type="file"> fields
    - Handle popups/dialogs (alert, confirm, prompt)

    Reading
    - Get a full accessibility snapshot of the page (text content with element references)
    - Evaluate JavaScript (async/await supported)
    - Capture network requests/responses (start, stop, list, detail)

    Visual
    - Screenshot (full page or specific element, PNG or JPEG)
    - Save as PDF (with paper format, landscape, scale, print background options)

    Examples of things you can ask me:
    - "Open gmail and send an email"
    - "Go to Twitter and like the top post"
    - "Take a screenshot of the current page"
    - "Fill out this form on [website]"
    - "Read the content of [url]"
    - "Search for [query] on Google"
    - "Click the accept button on [website]"
    - "Close all tabs and end the session"
