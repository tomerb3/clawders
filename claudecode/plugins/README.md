# 🎯 Plugins for Claude Code

Plugins extend Claude Code with additional commands, integrations, and capabilities.

---

## Installing Plugins

Use the built-in marketplace:

```bash
/plugin marketplace add <plugin-name>
```

---

## Must-Have Plugins

### 1. Superpowers

Smart planning, brainstorming, and context management.

```bash
/plugin marketplace add obra/superpowers-developing-for-claude-code
```

📺 [Video: Claw Code + Superpowers](https://www.youtube.com/watch?v=YJR8TfpPUrs)

### 2. Code Simplifier

Automatically rewrites complex code into simpler, more readable forms.

### 3. Context7

Keeps Claude Code up to date on the latest APIs, services, and libraries by providing fresh context — great for avoiding stale knowledge.
